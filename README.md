# Smart Resume Analyzer

This is a simple project I built using Streamlit. It analyzes a resume based on a selected job role and gives feedback.

## Features
- Upload resume (PDF)
- Select target role
- Shows match percentage
- Highlights missing skills
- Gives basic suggestions

## Tech Used
- Python
- Streamlit
- PyPDF2

## How to Run
1. Install:
   pip install streamlit and PyPDF2

2. Run:
   streamlit run app.py
